About Nutritionists API

The Project

  • This is a test project, part of the CodingFactory course and mandatory for graduating from this course.
  • The main purpose is to showcase the skills and knowledge acquired during the course.
  • More emphasis is given towards functionality and programming rather than aesthetic appeal and UI/UX.
  • The "nutritionist" aspect of the project was chosen as I am a Nutrition school graduate myself.

The stack

  • Angular 17 is used as the front-end, with emphasis towrads the latest updates including:
    • Standalone components
    • Signals and effects
    • {{ "@for" }}, {{"@if"}} etc. directives
    • Dependency injection using inject()
    • Material design
  • Django is used as back-end, utilizing Django Rest Framework and dj-rest-auth with JWT authentication.
  • SQLite is used as database due to ease of use during development.

Features

  • User:
    • Profile editing
    • Handling of measurements
    • Handling of diets
    • View the bio of assigned nutritionists - Not implemented yet
    • Get a chatGPT diet suggestion based on a set of measurements and preferrences - Not implemented yet
  • Nutritionist:
    • Profile editing
    • Handling of clients
    • Handling of clients' measurements and diets
    • Handling of diets
    • Linking clients to existing users as a way to make the measurements and diets available to the client - Not implemented yet
  • Staff
    • Profile editing - Not implemented yet
    • Listing of all users - Not implemented yet
    • Listing of all normal users - Not implemented yet
    • Listing of all nutritionists - Not implemented yet
